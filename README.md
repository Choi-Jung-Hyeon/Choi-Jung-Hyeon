## 🧭 About Me

> **"기술의 궁극적 목적은 사람들의 일상을 편리하고 안전하게 만드는 것"**

성균관대학교 소프트웨어학과에서 **백엔드와 AI**를 중심으로, 기획부터 인프라까지 서비스 전체 라이프사이클을 다루는 개발자입니다. 화려한 기술보다 **누구나 쉽게 쓸 수 있는 단단하고 따뜻한 서비스**를 지향합니다.

- 🏦 **삼성증권 YoungCreator 15기** — 리스크 리포트 엔진, OpenCode 연동 MCP, VVIP Asset Advisor Hub 개발
- ⚖️ **SKKUDING** — 성균관대 Online Judge [Codedang](https://codedang.com) 백엔드 엔지니어 (NestJS · Prisma 성능 최적화)
- 🔬 **LG AI 연구원 산학협력** — *Training-Free Detection of AI-Generated Images* 연구 팀장
- 🧠 **NLP Lab 학부연구생** — DPO/GRPO 논문 세미나, RAG · LLM 파이프라인, 페르소나 데이터 전처리
- 🏗️ **Hyscape SW 인턴십** — LLM API 연동 + 폐쇄망 NAS 기반 사내 데이터 자동화 파이프라인 구축 및 런칭
- 🎓 **졸업작품** — 행동재무학 기반 심리 지수 반영 [주가 예측 모델 & 웹 플랫폼](https://github.com/2026Graduation-Work/Stock_Prediction_v2)

## 💡 How I Work

- **기획 우선** — 코드를 짜기 전에 유저 관점의 요구사항을 분석하고 문서화합니다. 명확한 기획이 개발 속도를 좌우한다고 믿습니다.
- **실패는 성장의 데이터** — 에러 로그와 하드웨어 제약 앞에서 좌절 대신 문제 해결의 몰입을 즐깁니다.
- **솔루션 중심 코드 리뷰** — 오류 지적을 넘어 항상 적용 가능한 기술적 대안을 함께 제시합니다.
- **풀스택 시야** — 백엔드가 주 포지션이지만, 프론트엔드 · 기획 · 인프라까지 전체 흐름을 파악하고 병목을 방지합니다.
